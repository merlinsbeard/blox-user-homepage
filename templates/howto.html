User Guide

Hi and welcome to BLOX. This page contains everything you need to use your Blox media entertainment.

Initial Setup

You need the following in order to turn on Blox.

  • Monitor or TV with HDMI connection
  • 1 usb keyboard
  • 1 usb mouse


Blox needs these equipments to turn on and boot up. Afterwards, these equipement can be removed.

Getting Started

Blox's main page is where you can control and play your music, videos, and pictures. It also contains information on how to access your Blox's features.

Blox main page is already available at the startup of the Blox computer. You can access the Blox's user web site in other browser by using Blox's hostname or IP address.

Accessing Your BLOX using Hostname

Using hostname to access Blox's user webpage can only be used by Desktops and Laptops.

  1. Open your browser(Chrome, Firefox, Etc.)
  2. Type in the address bar: blox/ or blox.loca/

Accessing using IP address

You can access Blox's user website in smartphones or tablets using Blox's IP address.

  1. Open browser on your smartphone
  2. Type in the address bar: {{ dicts.IP }}

Your Files

One of the main features of blox is its File Server. Blox can store 1TB worth of Documents, Pictures, Videos, and Music.

There are 2 ways to access you file server using desktop/laptops or using your smartphone

Using Desktops or Laptops

The first one is by using Desktop or Laptop computers.

  1. On your Desktop press start
  2. Type \\blox\ or \\blox.local\
  3. Press start

Using Smartphone

You can also use Blox's file server by using smartphone.

  1. Download and Install android app ES File Explorer
  2. Open the app and go to Network > Lan Settings
  3. Tap New in the bottom
  4. In the host field put the IP address of blox {{ dicts.IP }}
  5. In the username and password put blox
  6. Click OK

Folder Structure

Blox file server contains the following:

  • Documents Folder where you can store your documents.
  • Music Folder where you store you Music collection.
  • Pictures Folder where you store your Picture collection.
  • Video Folder where you store your video collection sorted by: Movies, Music-Videos, Home-Videos, and others.
  • Public Folder this is shared to anyone

Your Videos and Pictures

Browse, Play, and Share your Videos and Photos. Watch or view your videos and photos on your smartphone, tablet, or Smart TV.

Blox uses Plex media server for its backend in Video and Photo serving. Plex lets you browse play, and share using Internet browser from your computers or desktops.

Storing Videos or Pictures

The first thing to do is to store your video or pictures. You can store your Videos and Pictures in the File Server

  1. Open your file server using your desktop or smartphone.

  2. If you want to store your Picture.
    1. Copy the picture you want to store
    2. Open the Picture folder in the file server
    3. Paste the picture
  3. If you want to store your Videos
    1. Copy the Video you want to store
    2. Open the Videos Folder in the File Server
    3. Paste the Video

Accessing your Media Server

Your Media Server can be access using: Internet browser in desktop and smartphones, and Plex Home Theater

Accessing through Browsers

  1. Go to Blox's User Home pagehttp://blox.local or http://{{ dicts.IP }}
  2. Click or tap the icon of Video

Access through Application

You can use Plex Home Theater in Blox if you dont want to use the browser version.

  1. On the Blox Computer click on the icon of Plex Home Theater

  2. If you are using Windows you can download Plex Home Theater

Android Phone for remote control

There is an app available that can remote control Plex Home Theater. You can also watch videos simultaneously.

  1. Download the Plex for Android

Watching your Videos and Pictures

Your Music

One of Blox's main feature is playing, organizing, and sharing music. Once your playlist is set up, Blox can automatically play it. You can also remotely control your music using your phone, laptop, or any other computer within your network.

Blox can be integrated to your audio system. Control it using your android phone, or any other computer within the network.

Adding Music

Blox uses the Music Folder in its File Server to find and play Music.

  1. Go to File Server and open Music Folder

  2. Store your Music here

Accessing your Music Server

In your Blox open the application Cantata. It is already configured to play any Music inside the folder of Blox.

  1. Click cantata

Access through Application

Music server can be accessed in any Dekstops or Laptops that has Cantata application installed.

  1. Download and install Cantata for windows
  2. Install Cantata
  3. Configure Cantata
    Name: Put any name here
    Host: {{ dicts.IP }}
    Port: 6600
    Password: blox
    Music Folder: Leave it as is
    click appliy and OK

Access through smartphone

Music server can be remotely controlled using Android Phone. You just need the MPDroid application.

  1. On your Android phone go to Blox Home Page
  2. Click on the Music Icon in smartphones
  3. Choose your smartphone from the application links. If you are using android phone choose android application.

    This will redirect you to the download page. Download and install the application.
  4. Open the application and go to settings

  5. tap connection settings

  6. Choose Wlan based connection

  7. Choose your wifi

  8. Here are the settings to put:
    Host: {{ dicts.IP }}
    Password: blox
    Leave everything else as is

  9. Press back until Now playing interface
