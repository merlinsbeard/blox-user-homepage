Android

Google Play store provides many applications that can remote control the Music Server (MPD).

  1. Go to playstore and download MPDroid
  2. Open the application and go to settings
  3. In the settings input the configuration

IOS

Just like android, IOS can also provides applications that can interact with the Music Server.

  1. Go to IOS store and download Mpod
  2. Open the application and go to settings
  3. In the settings input the configuration
